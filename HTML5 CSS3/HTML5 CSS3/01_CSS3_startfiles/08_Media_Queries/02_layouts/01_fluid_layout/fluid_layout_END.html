#content Fluid Layout - 01 Basic Fluid Layout

To show output

Fluid layout is based on a system of relative units instead of absolute pixels. This kind of layout has been around for a while now, and most of the designers use fluid grids created in percentage to achieve such layouts.

The idea is pretty simple: instead of giving the layout rigid width in pixels, we will give it relative ones in percentage. The fluid layout based websites usually take the whole browser width, hence the 100% in this example. http://www.onextrapixel.com/examples/responsive-design/example1_fluid.php
