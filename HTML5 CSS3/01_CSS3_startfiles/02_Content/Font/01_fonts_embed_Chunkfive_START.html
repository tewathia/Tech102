1. The @font-face rule allows for linking to fonts that are automatically activated when needed.
2. Multiple @font-face rules can be used to construct font families with a variety of faces.
3. Firefox, Chrome, Safari, and Opera support fonts of type .ttf (True Type Fonts) and .otf (OpenType Fonts).
4. Internet Explorer 9+ supports the new @font-face rule, but it only supports fonts of type .eot (Embedded OpenType).

@font-face
{
font-family: '';
src: url('.eot?') format('eot'),
url('.woff') format('woff'),
url('.ttf') format('truetype');
}

String Font Format Common extensions
"woff" WOFF (Web Open Font Format) .woff
"truetype" TrueType .ttf
"opentype" OpenType .ttf, .otf
"embedded-opentype" Embedded OpenType .eot
"svg" SVG Font .svg, .svgz
